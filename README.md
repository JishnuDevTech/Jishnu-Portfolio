# Jishnu Rahegaonkar Portfolio

[![License](https://img.shields.io/badge/License-MIT-green)](LICENSE)

A sleek, fully responsive personal portfolio website showcasing my projects, skills, and development journey. Designed with a modern, clean UI and optimized for both desktop and mobile users, it highlights my work as a front-end developer and provides easy navigation for potential employers or collaborators.

This portfolio website is a professional showcase of my work and skills in web development. It includes project highlights, skill sets, and contact information, all presented in a visually appealing, responsive layout.

- Easy navigation with sections for About, Projects, and Contact
